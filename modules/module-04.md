# Модуль 4. Фундаменты и основания

## Виды фундаментов

Основные типы фундаментов:

- **Ленточные** — для зданий с несущими стенами
- **Столбчатые** — для лёгких сооружений
- **Свайные** — для слабых грунтов
- **Плитные** — для тяжёлых зданий и слабых грунтов

## Подготовка основания

Перед устройством фундамента:

- Разметка осей здания
- Устройство песчаной или щебёночной подушки
- Уплотнение основания
- Установка опалубки

## Армирование

Армирование фундаментов выполняют для восприятия растягивающих напряжений:

- Арматура классов A400, A500, B500
- Защитный слой бетона — не менее 40 мм для фундаментов
- Шаг арматуры определяется расчётом

## Заливка бетона

- Марка бетона для фундаментов — не ниже B15
- Подвижность смеси П2–П4
- Уплотнение вибрированием
- Уход за бетоном в течение 7 суток
