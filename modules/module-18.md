# Модуль 18. Сдача объекта в эксплуатацию

## Этапы сдачи объекта

1. Предъявление объекта для приёмки
2. Проверка готовности объекта
3. Проведение комиссий по приёмке
4. Устранение замечаний
5. Подписание акта приёмки
6. Получение разрешения на ввод в эксплуатацию

## Комиссии по приёмке

- Государственная комиссия (для объектов, требующих разрешения)
- Рабочая комиссия (для остальных объектов)
- Специализированные комиссии (пожарная, санитарная, электрическая)

## Документы для сдачи

- Разрешение на строительство
- Проектная документация
- Исполнительная документация
- Сертификаты и паспорта на материалы
- Акты освидетельствования скрытых работ
- Журналы работ
- Результаты испытаний

## Устранение замечаний

- Составление дефектного ведомства
- Определение сроков устранения
- Повторная проверка устранённых замечаний
- Оформление актов об устранении замечаний
